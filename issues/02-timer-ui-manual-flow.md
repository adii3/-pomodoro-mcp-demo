`docs/prd-pomodoro-app.md`

## What to build

Build the timer screen for a single user with manual pomodoro controls. The user should be able to start, pause, resume, reset, complete a focus session, and start a break session, all through one clear interface.

## Acceptance criteria

- [ ] The page shows the active phase and remaining time
- [ ] The user can start, pause, resume, and reset the timer
- [ ] The user can manually complete a focus session
- [ ] The user can manually start a break timer
- [ ] Settings values are shown in the UI and can affect the timer
